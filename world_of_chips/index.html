What exactly is the Shockbox?

What does it do?

“Shockbox is a sensor that attaches to protective helmets and uses accelerometers to measure the effect of a head impact. This information is sent to the smartphone of a parent or a coach via Bluetooth, which allows them to make critical decisions like whether they should be taken off the field.”

Well done.

What exactly is a Raspberry Pi?

A credit card-sized computer that can be used in electronics projects, and for many of the things that your desktop PC can do.

What kind of electronics projects?

Pi Microwave

What's so great about a Pi Microwave?

  • Voice control
  • Barcode scanner to access online database for cooking times
  • Web-based interface for remote access
  • Auto-tweets when timer is done
